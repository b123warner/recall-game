# recall-game

A local co-op trivia game in the spirit of Jackbox Party Pack. Built with React + Vite + TypeScript + Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your key:

```
VITE_ANTHROPIC_API_KEY=your_key_here
```

## Tech Stack

- React + Vite
- TypeScript (strict mode)
- Tailwind CSS
- Zustand (game state)
- Open Trivia DB + Anthropic API (questions)
